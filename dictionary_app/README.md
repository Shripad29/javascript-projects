# 📚 Dictionary App

A simple yet functional **Dictionary App** built using **HTML**, **CSS**, and **JavaScript**. It fetches real-time definitions, examples, synonyms, and antonyms for any word using the free [dictionaryapi.dev](https://dictionaryapi.dev/) API.

---

## ✨ Features

- 🔍 Search any English word
- 📖 Displays:
  - Word & Part of Speech
  - Definition
  - Example (if available)
  - Synonyms
  - Antonyms
  - External "Read More" source
- 📱 Fully responsive UI for all screen sizes
- ⚡ Fast and clean user experience
- 💡 Error handling for invalid or undefined words

---


## 🛠️ Tech Stack

- HTML5
- CSS3 (Responsive & Styled using CSS variables)
- JavaScript (Vanilla)
- [dictionaryapi.dev](https://dictionaryapi.dev/) (Free Dictionary API)
