# 🖼️ Image Slider (Carousel)

A modern and responsive **Image Slider (Carousel)** built using **HTML, CSS, and JavaScript**.

This project allows users to navigate through images manually or automatically with smooth transitions.

---

## 🚀 Features

- Next and Previous navigation buttons
- Automatic image sliding (auto-play)
- Smooth transition animation
- Infinite looping of images
- Clean and responsive UI
- Beginner to intermediate level project

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Font Awesome Icons

---

## ⚙️ How It Works

1. Images are displayed in a horizontal slider.
2. Click **Next** or **Previous** buttons to navigate manually.
3. Images automatically slide every few seconds.
4. The slider loops infinitely when reaching the end.

---

## 💡 Key Concepts

- JavaScript DOM manipulation
- Event handling (click events)
- setTimeout for auto sliding
- CSS transitions and transforms
- Infinite looping logic

---

## 🧠 Learning Outcomes

- Building interactive UI components
- Working with image carousels
- Managing state (current image index)
- Creating smooth animations
