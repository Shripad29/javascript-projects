# 🖼️ Image Slider (Carousel)

A modern and responsive **Image Slider (Carousel)** built using **HTML, CSS, and JavaScript**.

This project allows users to navigate through multiple images using **next and previous controls**, along with an **auto-play feature** that automatically slides images after a fixed interval.

It demonstrates smooth UI transitions, interactive controls, and dynamic behavior using JavaScript.

---

## 🚀 Features

- Next and Previous navigation buttons  
- Automatic image sliding (auto-play)  
- Smooth transition animations  
- Infinite looping of images  
- Clean and responsive UI  
- Beginner to intermediate level project  

---

## 🛠️ Technologies Used

- HTML  
- CSS  
- JavaScript  
- Font Awesome Icons  

---

## ⚙️ How It Works

1. Images are arranged horizontally inside a container.  
2. Clicking **Next** or **Previous** buttons updates the current image.  
3. JavaScript shifts the image container using `transform: translateX`.  
4. Images automatically slide every few seconds using `setTimeout`.  
5. The slider loops infinitely when reaching the first or last image.
