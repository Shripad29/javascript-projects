# ⚖️ BMI Calculator

A simple and interactive **BMI (Body Mass Index) Calculator** built using **HTML, CSS, and JavaScript**.

It helps users calculate their BMI based on height and weight and shows their weight category instantly.

---

## 🚀 Features

- Calculate BMI instantly
- Displays weight category:
  - Underweight
  - Normal weight
  - Overweight
  - Obesity
- Clean and modern UI
- Beginner-friendly project
- Responsive design

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript

---

## ⚙️ How It Works

1. Enter your **height (in cm)**.
2. Enter your **weight (in kg)**.
3. Click on **Compute BMI** button.
4. The BMI value is calculated and displayed.
5. Your weight condition is shown below.


---

## 💡 BMI Formula

BMI is calculated using the formula:

BMI = weight (kg) / (height (m) × height (m))
