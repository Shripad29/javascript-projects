# 🔐 Secure Random Password Generator

A simple and secure **Random Password Generator** built using **HTML, CSS, and JavaScript**.

This project allows users to generate strong passwords instantly and copy them to the clipboard with one click.

---

## 🚀 Features

- Generate strong random passwords
- Includes letters, numbers, and special characters
- One-click copy to clipboard
- Clean and simple UI
- Copy notification alert
- Beginner-friendly JavaScript project

---

## 🛠️ Technologies Used

- HTML
- CSS
- JavaScript
- Font Awesome Icons

---

## ⚙️ How It Works

1. Click the **Generate** button.
2. A secure random password is created.
3. Click the **copy icon** to copy the password.
4. A notification appears confirming the password has been copied.

---


## 💡 Password Generation Logic

The password is generated using a string containing:

- Numbers
- Lowercase letters
- Uppercase letters
- Special characters

JavaScript selects random characters from this string to create a secure password.
